# Internal Sample Log

An Excel-based compliance automation tool designed to help cannabis licensees
track internal product samples and enforce Michigan regulatory limits
over a rolling 30-day period.

The system reduces manual METRC adjustments by generating structured CSV uploads
and flagging compliance risks before violations occur.

## Business Problem

Michigan cannabis regulations impose strict per-employee limits on internal
product samples. Tracking rolling 30-day totals across employees, product types,
and THC thresholds is error-prone when done manually.

This tool was designed to:
- Prevent regulatory violations
- Reduce manual METRC entry
- Improve audit readiness
- Increase operational efficiency

## Features

- Rolling 30-day calculations per employee
- License-type-specific compliance thresholds
- Separate handling of flower, concentrate, and infused products
- THC milligram aggregation
- Automatic compliance flags
- METRC-ready CSV output
